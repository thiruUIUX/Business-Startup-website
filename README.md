# Business Startup Website

A clean and simple 2-page business startup website designed using **HTML** and **CSS**. This project is ideal for showcasing basic web design skills and structuring a small business website.

## Features

- **Responsive Navigation Bar**: Links to Home, Pricing, and Contact Us pages.  
- **Visual Design**: Includes modern layout and appealing images to highlight business offerings.  
- **Easy to Customize**: Perfect template for small businesses to start their online presence.  

## Pages

1. **Home Page** – Introduction to the business with images and key information.  
2. **Pricing Page** – Displays different pricing plans clearly.  
3. **Contact Us Page** – Simple contact form or information to get in touch.  

## Tech Stack

- HTML  
- CSS
